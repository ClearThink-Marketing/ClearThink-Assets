# Completed ICP Example: Bell Curves (Education/Test Prep)

This is an example of a completed B2B ICP for an education services client. Notice how the generic B2B template was adapted with education-specific fields (Student Demographics, Cultural & Values Alignment) while keeping the core structure intact.

## 1. ICP Overview

| ICP Overview | |
| :---- | :---- |
| **Organization Type:** | Charter Networks/Schools |
| **One-Line Description:** | Companies that run charter schools on behalf of cities and states |
| **Why They're Ideal:** | They control multiple schools which means greater reach. More flexibility than public schools. You can do more with their students (sometimes) |
| **Strategic Priority:** | Primary |

## Key Adaptation Notes

- The template added a **Student Demographics Served** section because in education, the end user (students) significantly affects program design and cultural fit
- **Cultural & Values Alignment** was expanded because mission alignment often matters as much as program quality in education purchasing
- **Buying Behavior** includes education-specific procurement patterns (RFP, pilot preference, budget cycles tied to school years)
- The **Competitive Landscape** section includes specific named competitors (Kaplan, Princeton Review) with concrete weakness analysis

## Why This ICP Works

1. **Specific enough to target**: "Charter networks in urban areas with 5+ schools" is a media-buyable audience
2. **Clear disqualifiers**: "Schools unwilling to run diagnostics" — this prevents wasted sales cycles
3. **Economically justified**: Multi-school contracts with expansion potential = high LTV
4. **Actionable channels**: Conference attendance, professional associations, and referral networks are all reachable
5. **Connected to UVP**: The ICP's needs (underserved student populations, score improvement) directly map to Bell Curves' unique value proposition
