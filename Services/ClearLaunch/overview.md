# ClearLaunch

ClearLaunch is ClearThink's productized go-to-market strategy sprint — a structured 8-week process that replaces guesswork with a data-backed strategic foundation before a single dollar is spent on campaigns.

**$1,200 flat · 8 weeks · industry-agnostic**

For the full ClearLaunch positioning and how it fits into the ClearThink service portfolio, see the root [`Overview.md`](../../Overview.md).

## The 7-step process at a glance

1. **Onboarding** — Notion client portal setup + intake validation
2. **ICP Development** — Ideal Client Profile analysis
3. **Market Research** — 50-keyword landscape + competitive audit
4. **UVP Development** — Unique Value Proposition + positioning
5. **Offer Development** — Micro / Macro / Core three-tier offer ladder
6. **Channel Strategy** — Two-phase channel plan + projections + customer journey
7. **Metrics & Roadmap** — KPI framework + 90-day implementation plan *(not yet built — see [`../../CLEANUP.md`](../../CLEANUP.md))*

For step-level execution detail, see [`Process/overview.md`](./Process/overview.md). For full methodology, see the Blueprint above.
