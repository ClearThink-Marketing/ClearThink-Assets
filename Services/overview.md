# Services

This folder contains one subfolder per ClearThink service line. Each service folder holds that service's canonical reference material, delivery methodology, skills, and templates.

For company-level positioning and service pricing, see the root [`Overview.md`](../Overview.md).

---

## Adding a new service

When a new service line launches or needs repo content:

1. Create `Services/<Service-Name>/` folder (hyphens, no spaces)
2. Add an `overview.md` at the service root describing what's in the folder
3. Add supporting material (methodology docs, skills, templates) as needed
4. Update the table above to include the new service
